# Lights
Indicator L - Pin 0
Indicator R - Pin 1
Headlights - Pin 2
Brakes  lights - Pin 3

Motor Spped - Pin 5
Motor 1 Direction - Pin 6,7
Motor 2 Direction - Pin 7,8
